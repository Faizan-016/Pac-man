In this tutorial, I will show you how I coded Pac-Man using JavaScript and HTML5 canvas.
Throughout the tutorial, I will walk you through how I drew on the HTML5 canvas,
created the game loop, loaded the game map using a tilemap,
added click handlers to make Pac-Man move, created a simple algorithm
to move each ghost at random, detected collisions between Pac-Man and ghosts,
made Pac-Man eat the food pellets, added a running score, reset the game when Pac-Man collides with a ghost,
and moved onto the next level once Pac-Man eats all the food pellets in the current stage.
